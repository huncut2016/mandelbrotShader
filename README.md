# Mandelbrot-set
Mandelbrot set in [OpenFrameworks](https://openframeworks.cc/), with GLSL shaders.

## Formula
$z_0 = 0$

$z_{n+1} = z_{n}^2+c$

$M= \\{ c \in \mathbb {C} \ \mid \ x_{n}\not \rightarrow \infty \\} $


read more: [Mandelbrot-set](https://en.wikipedia.org/wiki/Mandelbrot_set)
